ITG.RegExps
===========

Наиболее востребованные регулярные выражения.

Версия модуля: **1.1.0**

Переменные и константы
----------------------

### reDomain

Регулярное выражение для поиска FQDN (полного доменного имени).

### reMailAddr

Регулярное выражение для поиска адресов электронной почты (SMTP).

### rePort

Регулярное выражение для выделения элемена пути в URL.

### reSchema

Регулярное выражение для выделения моникёра протокола.

### reSocket

Регулярное выражение для выделения сокета (FQDN:port).
